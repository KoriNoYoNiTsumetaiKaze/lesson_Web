1)Отключить интернет на время установки программы.
2)Установить пробную версию программы. 3)Запустить программу и согласиться начать пробное использование. 4)Закрыть программу. 5)Заменить файл amtlib.dll находящийся в папке с установленной программой на файл amtlib.dll находящийся в данной раздаче. Файл для 32 битной системы находится в папке 32-bit, а файл для 64 битной системы находится в папке 64-bit. 6)Включаем интернет. 7)Запускаем свежеустановленную программу. 8)Наслаждаемся жизнью! ВНИМАНИЕ: данная инструкция подходит для лечения любой программы из линейки Adobe CS6 1)Отключить интернет на время установки программы. 2)Установить пробную версию программы.
